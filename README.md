Steam & Silicon 🍳⚡
A futuristic EV charging diner with robot staff, cinema megascreens, and drone delivery. Built across 3 labs.

Lab 1 — HTML & CSS
Built a 3-page static site with a dark electric-blue theme, responsive navbar, hero section, stats bar, feature blocks, menu table, and reservations form. Learned the box model, Flexbox, CSS variables, and form structure.

Lab 2 — Bootstrap & JavaScript
Upgraded to a dynamic app. Menu items stored in a JavaScript array and rendered to the page via DOM manipulation. Added category filtering, a Bootstrap modal on card click, price formatting with Intl.NumberFormat, and JavaScript-only form validation with conditional show/hide sections. Learned forEach, addEventListener, preventDefault, and how data drives UI.

Lab 3 — Cart & Ordering
Built a full ordering system. Add to Cart saves items to localStorage with JSON.stringify. Cart page reads and parses the data, calculates subtotal, 8.25% tax, and total in JavaScript. Cancel Order shows a confirmation modal and redirects. Learned how data travels between pages and multi-page state management.

Stack: HTML5 · CSS3 · Vanilla JS · Bootstrap 5.3 · Font Awesome · Google Fonts
